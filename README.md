# API Test with POST Method in JS

This project is a simple example of how to send data from an HTML form to an API using the `POST` method in plain JavaScript.

<img src="git/demo1.png?raw=true"/>
<img src="git/demo2.png?raw=true"/>

## Structure

* `form.html`: Form used to input data.
* `post.js`: Script responsible for capturing form data and sending it via POST.

## How to use

1. Open the `form.html` file in a browser.
2. Fill in the form fields.
3. Click "Submit" to send the data to the API.

## Notes

* This project is only a learning example and does not include a backend.
* For local testing, you can use a simple server like:

  ```bash
  python -m http.server
  ```
